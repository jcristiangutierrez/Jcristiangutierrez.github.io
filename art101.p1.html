Drawing Application

For our first project in Art 101, we were tasked to develop our own drawing system on Processing, by creating our own functions with arguments. My inspiration for these brushes were optical illusions and trying to create something abstract. I wanted to create a brush that you can manipulate very easily, as well as making them transparent, so when crossing over another color they can create a new one. I really wanted to go for an abstract look on my drawings, I am pretty new to the style of abstract art so I tried to replicate my definition of abstract may or may not be correct. My favorite would have to be the the brush with the transparent rectangles because I was able to create these futuristic looking type snow mountain which reminded me of cover art for an old arcade game. see code
